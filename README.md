## Demo:

# Technologies:

- Microservice
- Framework: ExpressJs, React, FastApi
- Docker, Google Cloud Storage, Google Bigquery
- CI/CD: Github actions, neilify, vercel
- Database: MongoDB, Redis
- Other: Kafka, Nginx, Elasticsearch, Pytorch
